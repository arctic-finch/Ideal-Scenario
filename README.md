# Overview

**Ideal Scenario** is the first book on the Ideal Series, containing stories all around the universe of the series, as well as character introduction sheets. The book is primarily focused on romance and realizing fantasies, some less pure than others, in the form of a memory being recollected and told, so the order is pretty much optional. I hope you enjoy!
